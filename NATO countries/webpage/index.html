The 28 NATO countries


This cartography features the 28 nato members spatialized using geojson locations, with contextual figures on hovering.


Legend :

  • Size of the node is correlated to the country's population.

  • Color of the node is correlated to the country's opening rate for migrants : scaling from yellow (weak) to black (extensive).
